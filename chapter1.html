Chapter 1: C Arithmetic Operators

Welcome to the first chapter of our C tutorial! In this section, we'll explore Arithmetic Operators. These are fundamental symbols used to perform mathematical calculations like addition, subtraction, multiplication, and division on numerical values (constants and variables).

C provides several standard arithmetic operators that work much like they do in regular mathematics. Understanding these is crucial for writing programs that perform calculations. Let's look at the common ones. Click any paragraph to hear it read aloud.

Common Arithmetic Operators

1. Addition (`+`): Adds two operands together. 

#include 

int main() {
    int a = 10;
    int b = 5;
    int sum = a + b; // sum will be 15
    printf("Sum: %d\n", sum); // Output: Sum: 15

    float x = 2.5;
    float y = 1.5;
    float float_sum = x + y; // float_sum will be 4.0
    printf("Float Sum: %.1f\n", float_sum); // Output: Float Sum: 4.0
    return 0;
}

2. Subtraction (`-`): Subtracts the second operand from the first. 

#include 

int main() {
    int a = 10;
    int b = 5;
    int difference = a - b; // difference will be 5
    printf("Difference: %d\n", difference); // Output: Difference: 5
    return 0;
}

3. Multiplication (`*`): Multiplies two operands. 

#include 

int main() {
    int a = 10;
    int b = 5;
    int product = a * b; // product will be 50
    printf("Product: %d\n", product); // Output: Product: 50
    return 0;
}

4. Division (`/`): Divides the first operand by the second. Be careful with integer division! When both operands are integers, the result is also an integer, and any fractional part is discarded (truncated). 

#include 

int main() {
    int a = 10;
    int b = 5;
    int int_quotient = a / b; // int_quotient will be 2
    printf("Integer Quotient (10 / 5): %d\n", int_quotient); // Output: 2

    int c = 10;
    int d = 4;
    int truncated_quotient = c / d; // truncated_quotient will be 2 (not 2.5)
    printf("Truncated Quotient (10 / 4): %d\n", truncated_quotient); // Output: 2

    // To get a floating-point result, at least one operand must be a float/double
    float e = 10.0; // Make one a float
    int f = 4;
    float float_quotient = e / f; // float_quotient will be 2.5
    printf("Float Quotient (10.0 / 4): %.1f\n", float_quotient); // Output: 2.5
    return 0;
}

5. Modulus (`%`): Returns the remainder of an integer division. This operator only works with integer types. 

#include 

int main() {
    int a = 10;
    int b = 5;
    int remainder1 = a % b; // remainder1 will be 0 (10 divides evenly by 5)
    printf("Remainder (10 %% 5): %d\n", remainder1); // Output: 0 (Note: %% prints a literal %)

    int c = 10;
    int d = 4;
    int remainder2 = c % d; // remainder2 will be 2 (10 / 4 is 2 with a remainder of 2)
    printf("Remainder (10 %% 4): %d\n", remainder2); // Output: 2
    return 0;
}

These operators follow standard mathematical precedence rules (multiplication, division, and modulus typically happen before addition and subtraction). You can use parentheses `()` to control the order of operations explicitly, just like in algebra.
