Chapter 2: Basic Features

Now that you understand the basics from Chapter 1, let's look at the primary features. This section will introduce Tool A and Tool B. Click me to hear this introduction.

Tool A is used for performing Task Y. To use it, first select the item you want to modify, then click the Tool A icon in the (imaginary) toolbar.

Tool B complements Tool A by allowing you to adjust the parameters of Task Y. After using Tool A, the options for Tool B will become available in the properties panel.
